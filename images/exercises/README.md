# Imágenes de ejercicios

Poné acá los archivos `.jpg`/`.png` de las figuras que necesiten los ejercicios
(por ejemplo, diagramas de geometría). Después, en el panel admin → pestaña
"Ejercicios" → editar el ejercicio correspondiente, escribí la ruta exacta en
el campo "Imagen del ejercicio", por ejemplo:

```
images/exercises/b1-rectangulo.jpg
```

Hace falta volver a desplegar el sitio (push a GitHub) después de agregar una
imagen nueva para que quede disponible.
